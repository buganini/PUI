Callback Interface

All callback interfaces accept an event object as the first argument. Other args/kwargs will be passed through to the callback function. For example 

def button_cb(event, data):
    print(data)

Button("button").click(button_cb, 1)
will invoke 

button_cb(event, 1)
when the event is emitted.

Event Object

click

  • nothing for now

dblclick

  • .x, .y: double click position

mousedown

  • .x, .y: mouse down position

mouseup

  • .x, .y: mosue up position

mousemove

  • .x, .y: mouse move position

wheel

  • .x, .y: wheel position
  • .x_delta, .y_delta: pixel delta
  • .h_delta, .v_delta: angle delta

input

Triggered when typing in `TextField`
  • .value: text in edit buffer

change

Triggered when editing is finished in `TextField`
  • .value: new value
