Sizing

In following demos, containers come with 5px padding/gap and 3px colored border.

Rules

  • No unintended overflow.
  • An element could be categorized into 3 types:
    • Widget: A widget is a visible element, eg. a button, a label, etc.
    • Container: a container always grows to fit its content. (the specified size is essentially the min-size)
    • Scrollable: A scrollable can contain content larger than its size and can shrink on scrolling axes.
  • An element is in one of the following 3 sizing modes:
    • Static the element's size is specified or matches its content.
    • Weakly expanding the element is expanding with lower priority.
    • Strongly expanding the element is expanding with higher priority.
  • A container or a scroll is default to weakly expanding.
  • A weakly expanding element won't be expanding if:
    • there is any strongly expanding siblings.
    • it's the outmost container.
